## 💎 느낀점
- HTML/CSS를 많이 배우지도 않은 것 같은데도 벌써 할 수 있는 일들이 많아졌음
- 실습을 진행하다보니 `이 부분은 어떻게 설계하는걸까?`라는 의문이 끊이지를 않음
    - 확대/축소하였을 때 어떻게 반영이 되게 하였을까?
    - fixed는 어느 위치에 적용시키는걸까?
    - 각 block의 size는 어떻게 구분하는것이 효율적일까?
- 계속 공부하다보면 알게되겠지!
